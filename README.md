# 📖 Hello Rails React <a name="about-project"></a>


**Hello Rails React** is a project that employs Ruby on Rails app with PostgreSQL database, React and Redux to practice creating one application using for both front end and back end. This is rendering random greeting to the user.

## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 gem install rails
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/sahlminkok/hello-rails-react.git
```

### Install

Install this project with:

```sh
  cd hello-rails-react
  gem install
```

### Usage

To run the project, execute the following command:


```sh
  ./bin/dev
```
